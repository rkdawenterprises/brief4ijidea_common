brief4ijidea

Brief Editor Emulator for IntelliJ IDEA. This IntelliJ IDEA plugin adds key bindings and functionality that attempts to emulate many features of the original BRIEF MS-DOS application.

Installation

  • Using IDE built-in plugin system:

    Settings/Preferences > Plugins > Marketplace > Search for "brief4ijidea" > Install Plugin

  • Manually:

    Download the latest release and install it manually using Settings/Preferences > Plugins > ⚙️ > Install plugin from disk...
